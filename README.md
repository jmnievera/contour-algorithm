
# Contour Algorithm

This project addresses efficient calculation of the contour of a region of white pixels
on a black background, in both a high level language (C) and assembly (RISC-V).




## Lessons Learned

When building this project, I was able to act the developer and the compiler. I started with a high-level implementation of the algorithm in C. Upon completion, I ran test-scripts to ensure the accuracy of my algorithm. From that point, I was able to implement my algorithm into RISC-V. I then optimized my algorithm to reduce dynamic memory usage.
